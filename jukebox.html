John Sowell's Portfolio -- Juke Box

Juke Box, written by John Sowell, is a desktop GUI application, made to be like an actual juke box. But this user interface does more than just play songs. The user can also do a database search for certain songs, based on different criteria, and the user can also create and play playlists from selected songs. This juke box application currently contains over 4000 songs. Below is the main screen, when it is first opened. The first list box contains the complete list of songs in the juke box database. The second list box contains the current playlist, and the third list box contains the queue of selected songs.
